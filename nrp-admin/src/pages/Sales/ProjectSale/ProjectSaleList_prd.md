# 项目列表 - 产品需求文档 (PRD)

## 页面基本信息

| 属性 | 值 |
|------|-----|
| 页面名称 | 项目列表 |
| 页面路径 | `/sales/project-sale/list` |
| 所属模块 | 销售中心 - 项目销售 |
| 页面类型 | 列表页 |

---

## 一、功能概述

该页面主要用于项目销售列表的查询、浏览和操作。
- 支持多条件筛选项目信息
- 表格展示项目详细信息
- 支持查看、编辑项目操作
- 支持新增项目操作
- 分页展示便于大量数据浏览
- 目标用户是销售人员或销售管理人员

---

### 2.2 筛选功能

| 总条数显示 | 显示"共 X 条" |

## 三、交互设计

### 3.1 页面布局

- **筛选区域**：页面顶部，横向排列的筛选表单
- **列表区域**：中间区域，表格展示数据
- **分页区域**：页面底部固定，显示分页控件

### 3.2 表格交互

- 表头固定，不随内容滚动
- 表格内容区域可上下滚动
- 横向滚动查看更多列（总宽度1600px）
- 固定左侧序号、项目编号列和右侧操作列

### 3.3 按钮行为

| 按钮 | 行为 |
|------|------|
| 查询 | 验证表单后，按筛选条件查询，重置到第1页 |
| 重置 | 清空筛选表单，恢复默认状态 |

